###A cli app written to crawl and store social media 
profiles with slight search capabilities

As of now, support for WhatsApp, Telegram, Instagram and LinkedIn
profiles are added. More platforms will only be added as per requested.

You are free to use this code in any non-harmful or invasive way you wish.

WARNING! The current version may have slight issues as I did not have the time
or patience to test the new additions to the LinkedIn crawler.
